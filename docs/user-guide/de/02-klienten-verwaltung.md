# Klienten-Verwaltung

Diese Anleitung behandelt die Grundlagen der Verwaltung Ihrer Klienten in MyoFlow, vom Hinzufügen neuer Klienten bis zur sicheren Aufbewahrung ihrer Daten.

## Einen neuen Klienten erstellen

So fügen Sie einen neuen Klienten zu Ihren Unterlagen hinzu:

1.  Navigieren Sie im Hauptmenü zum Bereich "Klienten".
2.  Klicken Sie auf die Schaltfläche "Neuer Klient".
3.  Füllen Sie das Klientinformationsformular aus. Die folgenden Felder sind für die Einhaltung der österreichischen Rechnungslegungsstandards erforderlich:
    *   Vollständiger Name
    *   E-Mail
    *   Telefonnummer
    *   Vollständige Postanschrift (Straße, Postleitzahl, Stadt, Land)

[Screenshot: Formular für neuen Klienten mit Adressfeldern]

## Ein Klientenprofil anzeigen

Um das Profil eines Klienten anzuzeigen, klicken Sie einfach auf dessen Namen in der Klientenliste. Die Profilansicht zeigt alle Informationen des Klienten, einschließlich Kontaktdaten, Terminhistorie und alle von Ihnen hinzugefügten Notizen.

[Screenshot: Klientenprofilansicht]

## Klientinformationen bearbeiten

Sie können die Informationen eines Klienten jederzeit aktualisieren, indem Sie zu seinem Profil navigieren und auf die Schaltfläche "Bearbeiten" klicken.

## Klientennotizen hinzufügen

Sie können einem Klientendatensatz mit einem Zeitstempel versehene Notizen hinzufügen, um wichtige Informationen festzuhalten.

1.  Gehen Sie im Profil des Klienten zum Abschnitt "Notizen".
2.  Klicken Sie auf "Notiz hinzufügen", um einen neuen Eintrag zu erstellen.

## Klienten löschen

Um einen Klienten zu löschen, gehen Sie zu seinem Profil und klicken Sie auf die Schaltfläche "Löschen". Ein Bestätigungsdialog wird angezeigt, um versehentliches Löschen zu verhindern.

## Sicherheitshinweis

MyoFlow verwendet eine Verschlüsselung auf Feldebene (über libsodium), um sensible Klientendaten wie Gesundheitskennzeichen und medizinische Notizen zu schützen und sicherzustellen, dass diese Informationen vertraulich und sicher bleiben.

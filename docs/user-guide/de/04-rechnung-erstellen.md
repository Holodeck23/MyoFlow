# Rechnung Erstellen

Diese Anleitung beschreibt den Prozess der Erstellung und Verwaltung von professionellen, österreichkonformen Rechnungen mit MyoFlow.

## Eine Rechnung erstellen

So erstellen Sie eine neue Rechnung:

1.  Navigieren Sie zum Bereich "Rechnungen".
2.  Klicken Sie auf die Schaltfläche "Neue Rechnung".
3.  Wählen Sie den Klienten und das Leistungsdatum aus. Die Datumsauswahl lässt keine zukünftigen Daten zu, um die Einhaltung der österreichischen Vorschriften zu gewährleisten.

[Screenshot: Neues Rechnungsformular mit Datumsauswahl]

## Rechnungspositionen hinzufügen

Für jede Rechnung können Sie mehrere Positionen hinzufügen. Für jede Position müssen Sie Folgendes angeben:

*   Eine Beschreibung der Dienstleistung oder des Produkts.
*   Die Menge.
*   Den Stückpreis.
*   Den anwendbaren Mehrwertsteuersatz.

[Screenshot: Tabelle der Rechnungspositionen]

## Auswahl des Mehrwertsteuersatzes

MyoFlow unterstützt verschiedene Mehrwertsteuerstatus, um Ihren geschäftlichen Anforderungen gerecht zu werden:

*   **Kleinunternehmer:** Für Kleinunternehmen, die von der Mehrwertsteuer befreit sind.
*   **10 %, 13 % und 20 %:** Standardmäßige österreichische Mehrwertsteuersätze.

## Fortlaufende Rechnungsnummerierung

MyoFlow vergibt automatisch eine fortlaufende Rechnungsnummer im österreichischen Standardformat `JJJJ-NNN` (z. B. `2025-001`), um sicherzustellen, dass Ihre Unterlagen geordnet und konform sind.

## PDF-Rechnungen herunterladen

Sobald eine Rechnung fertiggestellt ist, können Sie eine professionelle, österreichkonforme PDF-Version für Ihre Unterlagen oder zum Versenden an Ihre Klienten erstellen. Klicken Sie einfach auf die Schaltfläche "PDF herunterladen" auf der Seite mit den Rechnungsdetails.

[Screenshot: Erzeugte PDF-Rechnung]

## Rechnungsverlauf anzeigen

Der Bereich "Rechnungen" bietet einen vollständigen Verlauf aller Ihrer Rechnungen, der nach Status, Klient oder Betrag durchsucht und sortiert werden kann.

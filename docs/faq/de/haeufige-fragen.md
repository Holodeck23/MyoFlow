# Häufig gestellte Fragen (FAQ)

## Konto & Einrichtung
**F: Wie melde ich mich bei MyoFlow an?**
A: Sie können sich mit Ihrer E-Mail-Adresse und einem Passwort anmelden oder Ihr Google-Konto verknüpfen.

**F: Welche Sprachen werden unterstützt?**
A: MyoFlow ist auf Deutsch (Österreich) und Englisch verfügbar.

**F: Kann ich meine Spracheinstellung ändern?**
A: Ja, Sie können die Sprache jederzeit über den Schalter in der oberen rechten Ecke jeder Seite wechseln.

**F: Was muss ich während des Onboarding-Assistenten tun?**
A: Der 3-stufige Assistent führt Sie durch die Einrichtung Ihrer wesentlichen geschäftlichen und beruflichen Daten.

## Klienten
**F: Warum muss ich die vollständigen Adressen meiner Klienten eingeben?**
A: Eine vollständige Klientenadresse ist eine gesetzliche Anforderung für die österreichische Rechnungskonformität.

**F: Wie werden meine Klientendaten gesichert?**
A: Sensible Klientendaten wie Gesundheitsnotizen werden mit einer Verschlüsselung auf Feldebene unter Verwendung von libsodium geschützt.

**F: Kann ich einen Klienten löschen?**
A: Ja, Sie können einen Klienten auf seiner Profilseite löschen. Ein Bestätigungsdialog verhindert ein versehentliches Löschen.

**F: Wo kann ich Notizen zu einem Klienten hinzufügen?**
A: Sie können in jedem Klientenprofil mit einem Zeitstempel versehene Notizen hinzufügen.

## Rechnungen
**F: Warum kann ich kein zukünftiges Leistungsdatum auf einer Rechnung auswählen?**
A: Das österreichische Gesetz schreibt vor, dass Rechnungen für bereits erbrachte Leistungen ausgestellt werden müssen. Die Datumsauswahl verhindert die Auswahl zukünftiger Daten, um die Konformität sicherzustellen.

**F: Welches Rechnungsnummernformat verwendet MyoFlow?**
A: Rechnungen werden fortlaufend im in Österreich üblichen Format `JJJJ-NNN` (z. B. `2025-001`) nummeriert.

**F: Wie lade ich eine PDF-Rechnung herunter?**
A: Sie können das PDF herunterladen, indem Sie auf der Seite mit den Rechnungsdetails auf die Schaltfläche "Herunterladen" klicken.

**F: Kann ich eine Rechnung für einen Klienten erstellen, der keinen Termin hatte?**
A: Ja, Sie können Rechnungen für jeden Klienten unabhängig vom Terminkalender erstellen.

## Steuern & Konformität
**F: Was ist die Kleinunternehmerregelung?**
A: Es handelt sich um eine österreichische Steuervorschrift, die Unternehmen mit einem jährlichen Bruttoumsatz von unter 55.000 € von der Umsatzsteuer befreit.

**F: Wann sollte ich Umsatzsteuersätze auf meinen Rechnungen verwenden?**
A: Sie sollten die Umsatzsteuer anwenden, wenn Ihr Jahresumsatz den Kleinunternehmer-Schwellenwert von 55.000 € übersteigt oder wenn Sie sich freiwillig für die Umsatzsteuerpflicht entschieden haben.

**F: Was ist der Unterschied zwischen den Umsatzsteuersätzen 10 %, 13 % und 20 %?**
A: Der richtige Satz hängt von der Art der Dienstleistung oder des Produkts ab. Die meisten therapeutischen Dienstleistungen fallen unter den 10%-Satz. Bitte konsultieren Sie unseren österreichischen Steuerleitfaden oder einen Steuerberater für Details.

**F: Unterstützt MyoFlow die RKSV (Registrierkassenpflicht)?**
A: Noch nicht. RKSV-konforme Funktionen sind für ein zukünftiges Update geplant. Wir empfehlen, einen Steuerberater zu konsultieren, wenn Ihr Umsatz sich dem Schwellenwert von 15.000 € nähert.

## Einstellungen
**F: Wo kann ich mein Firmenlogo zu Rechnungen hinzufügen?**
A: Sie können Ihr Logo unter "Einstellungen" → "Rechnungsbranding" hochladen.

**F: Wie richte ich meine Reisetarife ein?**
A: Konfigurieren Sie Ihren Basisstandort, Ihre Transportmethode und Ihre Tarife unter "Einstellungen" → "Reisen".

**F: Was bedeutet das Widget "Profilvervollständigung" auf dem Dashboard?**
A: Es zeigt den Prozentsatz der erforderlichen Geschäftsinformationsfelder an, die Sie in Ihren Profileinstellungen ausgefüllt haben. Ein vollständiges Profil ist für eine konforme Rechnungsstellung erforderlich.

**F: Kann ich meine Berufsbezeichnung ändern?**
A: Ja, Sie können Ihren Berufstitel (z. B. Heilmasseur) unter "Einstellungen" → "Profil" aktualisieren.
